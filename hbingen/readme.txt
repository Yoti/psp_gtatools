[hbingen.exe]
	Программа для автоматического создания файла
	AT3HEADER.BIN* на основе своих радиостанций.
	* - имя файла может варьироваться
	Необходимо поместить exe файл и папку "base"
	в папку, содержащую ВСЕ свои at3 станции.
	После запуска файл [код игры].bin необходимо
	переименовать так, как он хранится в игре.

	Использование:
		hbingen.exe [код игры]
	Пример:
		hbingen.exe ULES00502

	R1: поддержка ULES00502 (VCS/PSP)
	R2: расширяемая база данных игр
