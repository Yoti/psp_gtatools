[rawscan.exe]
	Программа для автоматического извлечения всех VAG
	из RAW контейнера.
	Необходимо запустить программу с параметром "имя
	raw файла" или перетащить RAW на иконку программы.
	После чего в той же папке где RAW будет создана
	папка "файл_raw.dir", а в ней - VAG файлы. Также
	будет создан необходимый файл "файл_raw.ols".
